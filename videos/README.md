# Hero Video Assets

Put your personal hero videos here:

- `hero-loop.mp4` for desktop and tablet
- `hero-loop-mobile.mp4` for mobile portrait screens

Recommended export settings:

- MP4 / H.264
- muted loop, 6-12 seconds
- desktop: 1920x1080 or 2160x1440
- mobile: 1080x1920
- keep the subject centered with some room for oversized typography
- use a light gray / white studio background with orange accent elements for the closest match

If these files are missing, the site falls back to the generated placeholder portrait in `public/images/hero-portrait.svg`.
